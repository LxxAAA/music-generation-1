# Music-Generation

## Requirements

- Python 3.6
- Keras 2.1.6
- Tensorflow (used as Keras's backend) 1.8.0
- Music21 5.1.0

The two last requirements are only necessary if you want to be able to listen to MIDI files directly with Python.

- midi2audio 0.1.1
- FluidSynth 1.1.1 (and at least one audio font)
